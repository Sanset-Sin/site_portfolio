ПРОЕКТ: Одностраничное портфолио-резюме

Состав:
- index.html
- style.css
- script.js
- assets/logo.svg
- assets/profile.svg

Что реализовано:
1. Подход Mobile First
2. Адаптивная одностраничная структура
3. Современный монохромный дизайн (2–3 основных цвета)
4. Собственный логотип в SVG
5. JavaScript:
   - мобильное меню
   - анимация появления секций при прокрутке
   - фильтрация проектов
   - подсветка активного раздела в меню
   - переключение темы
   - кнопка "наверх"
6. Footer с указанием нейросети, времени и JS-функционала
